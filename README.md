# Favorite-Food-PZ-Mod
This is a mod for Project Zomboid - "Favorite Food"

This mod will allow you to add a favorite food to your character on the Trait page in character creation. 
This favorite food will override the stats of that food to give higher positive stats (hunger, happiness, etc) and remove negative stats (boredom, stress, etc)

Icons for the traits are used from the base game and are owned by Indie Stone.
